# Безопасность

## Сообщить об уязвимости

Если вы обнаружили уязвимость, пожалуйста, не создавайте публичный issue. Опишите проблему по контакту из раздела «Поддержка» в README или в описании репозитория.

Данные приложения хранятся локально и шифруются на устройстве пользователя. Исходный код не содержит паролей или ключей доступа к внешним сервисам.
